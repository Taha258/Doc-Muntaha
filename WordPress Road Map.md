# 🚀 30 Days WordPress Job-Ready Plan

## 🟢 Week 1 — Foundation Strong (Day 1–7)

👉 Goal: WordPress ka complete base clear ho

### 📌 Learn:

* WordPress install (XAMPP / LocalWP)
* Dashboard complete samajhna
* Posts vs Pages difference
* Themes install + activate
* Plugins install/use
* Elementor basics

### 🛠 Practice:

* 1 simple blog website banao
* 1 business homepage Elementor se design karo

### 🎯 Output:

✔ 2 basic websites ready

---

## 🔵 Week 2 — Theme + Customization (Day 8–14)

👉 Goal: Thora developer level ka kaam

### 📌 Learn:

* Theme structure:

  * header.php
  * footer.php
  * index.php
* Child theme concept
* Custom menus + widgets
* Contact form plugin (WPForms)

### 🛠 Practice:

* 1 business website full customize karo (without demo import)
* Header/footer custom design karo

### 🎯 Output:

✔ 1 professional business website

---

## 🟡 Week 3 — PHP + WordPress Core (Day 15–21)

👉 Goal: Developer mindset

### 📌 Learn PHP basics:

* variables, arrays, loops
* functions

### 📌 WordPress Core:

* hooks (actions + filters)
* functions.php use

### 🛠 Practice:

* Custom function banao:

  * custom message display
  * shortcode create karo
* Simple plugin banao:

  * “Hello World Plugin”

### 🎯 Output:

✔ 1 custom plugin + shortcode

---

## 🔴 Week 4 — Real Job Skills (Day 22–30)

👉 Goal: Interview + internship ready

### 📌 Learn:

* WooCommerce basics
* Website speed optimization
* SEO basics (Yoast plugin)
* Hosting + domain setup

### 🛠 Final Projects (IMPORTANT):

1. Ecommerce website (WooCommerce)
2. Portfolio website (your CV style)
3. Landing page (high conversion design)

---

# 💼 Final Output (End of 30 Days)

Tumhare paas hoga:

✔ 4–5 live websites
✔ 1 plugin (basic)
✔ Elementor + custom theme knowledge
✔ WooCommerce understanding
✔ Portfolio ready
